# aruaru-web

aruaru-web is the main Web product.

## Role

aruaru-web integrates:

- aruaru-ai
- second KUSANAGI features
- project dashboard
- SFTP deployment
- domain and subdomain management
- HTTPS monitoring and maintenance
- learning and certification features

## First Boot by IP Address

When aruaru-web is first opened by IP address, it should guide the user to:

1. Register a main domain.
2. Register subdomains.
3. Register additional sites.
4. Check whether HTTPS is already enabled.
5. Automatically set up HTTPS when possible.
6. Start periodic health reports.

## User Levels

aruaru-web must support:

- VPS beginners with guided UI.
- Experienced infrastructure engineers with advanced settings and logs.
