# Product Structure

## aruaru-core

Non-AI central plugin collection.

Examples:

- config
- authentication
- logging
- monitoring
- database routing
- backup
- deployment hooks

## aruaru-ai-core

AI-related central plugin collection and the core AI control platform for aruaru-ai.

Examples:

- AI routing
- AI scoring
- AI benchmark
- AI cost optimization
- quality gate
- hardware diagnosis

## aruaru-web

The main Web platform. aruaru-ai is included inside aruaru-web.

## aruaru-desktop

Desktop application with Windows installer target.

## aruaru-mobile

Mobile application target for Android and iPhone.
