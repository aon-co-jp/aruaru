# Hardware Diagnosis

Before recommending local AI or aruaru-llm, aruaru-ai should scan the user's hardware.

## Scan Items

- CPU model
- core count / thread count
- system memory
- GPU model
- VRAM
- disk type and free space
- OS
- network speed
- thermal and power risks

## Output

The app should explain:

- whether aruaru-llm alone is realistic
- whether a selected local LLM is realistic
- whether external AI is better
- whether aruaru-llm + external AI is recommended
- what hardware upgrade would help
- approximate used/new price range when data is available
- expected performance improvement as an estimate

## Low-Spec Example

For a GT730-class PC, full local LLM development is not recommended. External AI or cloud AI should be used for heavy development tasks.
