# aruaru-ai-core

aruaru-ai-core is the AI control platform for aruaru-ai.

## Supported AI Candidates

- ChatGPT
- Claude Opus
- Grok
- Gemini
- DeepSeek
- Qwen
- local LLM
- aruaru-llm

## Selection Modes

- Auto
- Manual
- Multi-AI compare
- aruaru-llm + external AI

## Responsibilities

- analyze user tasks
- choose suitable AI
- estimate cost
- estimate hardware capability
- compare strengths and weaknesses
- run quality gates
- prevent endless bug-fix loops
