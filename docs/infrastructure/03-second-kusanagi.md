# Second KUSANAGI Design

aruaru-web should include second KUSANAGI style VPS automation.

## Target Users

- VPS beginners
- web developers
- infrastructure engineers
- rental server users

## Features

- first boot from IP address
- domain registration
- subdomain registration
- multi-site registration
- HTTPS status monitoring
- automatic HTTPS setup when possible
- certificate expiration monitoring
- automatic renewal
- periodic status reports
- failure alerts
- rollback support

## Important Policy

The system should make common VPS operations understandable and safe, while still allowing expert-level manual control.
