# aruaru-sftp

aruaru-sftp manages SFTP / SSH deployment to rental VPS servers.

## Features

- IP / host registration
- username registration
- password registration
- private key registration
- public key upload
- connection test
- remote directory creation
- file upload
- deployment log

## Key Management

- private keys must not be logged
- passwords must not be stored as plain text
- public keys may be uploaded to servers
- Windows Credential Manager or encrypted local storage should be used when possible
