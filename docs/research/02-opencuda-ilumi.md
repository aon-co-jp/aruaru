# OpenCUDA / iLUMI Integration

OpenCUDA / iLUMI is a related research direction for cross-platform GPU, AI, rendering, and computation abstraction.

## Target Platforms

- Windows / DirectX
- macOS
- Linux
- Android
- iPhone / iPad

## Relationship with aruaru-ai

aruaru-ai and OpenCUDA / iLUMI should have a strong integration path for:

- local LLM research
- GPU acceleration
- game-style learning
- 3D education apps
- cross-platform development
- hardware compatibility research

## Status

This is a research-stage module and should not block MVP development.
