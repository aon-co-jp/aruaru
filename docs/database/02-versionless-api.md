# Versionless API

aruaru should reduce API version breakage through a Versionless API concept.

## Rule

REST API is not the default communication policy.

aruaru prioritizes:

- WunderGraph Cosmo
- graph-based composition
- schema checks
- compatibility checks
- Versionless API design

## Goal

Minimize breaking changes between aruaru-web, aruaru-desktop, aruaru-mobile, and backend services.
