# MVP Scope

MVP v0.1 must create the smallest useful version of aruaru.

## Included

- aruaru-web basic dashboard
- project registration
- AI manual selection
- simple AI auto-selection
- hardware diagnosis
- SFTP connection registration
- server directory creation
- simple file upload
- PostgreSQL connection design
- domain / subdomain registration design
- HTTPS monitoring design

## Not Included Yet

- full aruaru-mobile native app
- full Windows installer
- full distributed DB
- full Git-style DB history
- full aruaru-llm research implementation
- 3D online learning game
- full automatic AI benchmark crawler

These are not deleted. They are moved to later phases.
